    Orthogonal projection in Euclidean affine spaces #

    This file defines orthogonal projection onto an affine subspace in a Euclidean affine space, and reflection of a point in an affine subspace.

    Main definitions #

    • EuclideanGeometry.orthogonalProjection is the orthogonal projection of a point onto an affine subspace.

    • EuclideanGeometry.reflection is the reflection of a point in an affine subspace.

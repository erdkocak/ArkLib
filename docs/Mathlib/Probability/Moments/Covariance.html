    Covariance #

    We define the covariance of two real-valued random variables.

    Main definitions #

    • covariance: covariance of two real-valued random variables, with notation cov[X, Y; Œº]. cov[X, Y; Œº] = ‚à´ œâ, (X œâ - Œº[X]) * (Y œâ - Œº[Y]) ‚àÇŒº.

    Main statements #

    Notation #

    • cov[X, Y; Œº] = covariance X Y Œº
    • cov[X, Y] = covariance X Y volume

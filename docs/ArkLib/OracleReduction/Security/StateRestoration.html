    State-Restoration Security Definitions #

    This file defines state-restoration security notions for (oracle) reductions.

    def Prover.StateRestoration.Soundness {ι : Type} (oSpec : OracleSpec ι) (StmtIn : Type) {n : ℕ} (pSpec : ProtocolSpec n) :
    Type 1

    The type for the state-restoration prover in the soundness game.

    Such a prover has query access to challenge oracles that can return the i-th challenge, for all i : pSpec.ChallengeIdx, given the input statement and the transcript up to that point. It returns an input statement, and a full transcript of interaction.

    This is different from the state-restoration prover type in the knowledge soundness game, which additionally needs to output an output witness.

        Type 1

        The type for the state-restoration prover in the knowledge soundness game.

        Such a prover has query access to challenge oracles that can return the i-th challenge, for all i : pSpec.ChallengeIdx, given the input statement and the transcript up to that point. It returns an input statement, a full transcript of interaction, and an output witness.

        Note that the output witness is an addition compared to the state-restoration soundness prover type.

    Duplex Sponge Fiat-Shamir #

    We define the (multi-round) Fiat-Shamir transformation using duplex sponges.

    NOTE: we currently do not define the salt explicitly. The salted version of the transform can be obtained via applying the transform to the protocol with the added salt as the first message.

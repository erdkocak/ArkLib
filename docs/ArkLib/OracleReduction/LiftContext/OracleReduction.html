    Lifting Oracle Reductions to Larger Contexts #

    This file is a continuation of LiftContext/Reduction.lean, where we lift oracle reductions to larger contexts.

    The only new thing here is the definition of the oracle verifier. The rest (oracle prover + security properties) are just ported from LiftContext/Reduction.lean, with suitable conversions.

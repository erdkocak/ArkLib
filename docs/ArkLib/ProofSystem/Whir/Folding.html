    Folding #

    This file formalizes the notion of folding univariate functions and lemmas showing that folding preserves list decocidng, introduced in the [Section 4 of the WHIR paper][todo: ArkLib bibliography].

    Implementation notes (corrections from paper) #

    • Theorem 4.20: -- proximity generators should be defined for C^(0),...,C^(k) in place of C^(1),...,C^(k) -- \delta \in (0, 1 - max_{i \in [0,k]} {....}) in place of \delta \in (0, 1 - max_{i \in [k]} {....})

    • Theorem 4.20 holds for l = 2 as can be seen with BStar(..,2) and errStar(..,2,..) and so Gen(l,alpha) = {1, alpha,...., alpha^{l-1}} also corresponds to l = 2 and not for a generic l.

    • Lemmas 4.21,4.22,4.23 -- these lemmas refer to the specific case when k set to 1, so it's safe to use the hypothesis 1 ≤ m

    References #

    • G Arnon, A Chiesa, G Fenzi, and E Yogev, [WHIR: Reed–Solomon Proximity Testing with Super-Fast Verification][todo: ArkLib bibliography] Freely available at https://eprint.iacr.org/2024/1586

                        class Fold.GenMutualCorrParams {F : Type} [Field F] [Fintype F] [DecidableEq F] {ι : Type} (S : Finset ι) (φ : ι ↪ F) (k : ℕ) :
                        Type 1

                        The GenMutualCorrParams class captures the necessary parameters and assumptions to model a sequence of proximity generators for a set of smooth ReedSolomon codes. It contains the following:

                        for i ∈ [0,k] :

                        • inst1, inst2, inst3: typeclass instances required to operate on ι^(2ⁱ) (finiteness, nonemptiness, and decidable equality).
                        • φ_i: per-round embeddings from ι^(2ⁱ) into F.
                        • inst4: smoothness assumption for each φ_i.
                        • Gen_α i: the proximity generators wrt the generator function Gen(parℓ,α) : {1,α,α²,..,α^{parℓ-1}} defined as per hgen for code Cᵢ
                        • inst5, inst6 : typeclass instances denoting finiteness of parℓ underlying Gen_αᵢ and parℓ_type
                        • BStar, errStar: parameters denoting proximity and error thresholds per round.
                        • h: main agreement assumption, stating that each Gen_α satisfies mutual correlated agreement for its underlying code.
                        • hcard, hcard' : |Gen_αᵢ.parℓ| = 2 and |parℓ_type| = 2
                        • m : ℕ
                        • inst1 (i : Fin (k + 1)) : Fintype (BlockRelDistance.indexPowT S φ ↑i)
                        • inst2 (i : Fin (k + 1)) : Nonempty (BlockRelDistance.indexPowT S φ ↑i)
                        • inst3 (i : Fin (k + 1)) : DecidableEq (BlockRelDistance.indexPowT S φ ↑i)
                        • φ_i (i : Fin (k + 1)) : BlockRelDistance.indexPowT S φ ↑i ↪ F
                        • inst4 (i : Fin (k + 1)) : Smooth (φ_i i)
                        • parℓ_type : Fin (k + 1) → Type
                        • inst5 (i : Fin (k + 1)) : Fintype (parℓ_type S φ i)
                        • exp (i : Fin (k + 1)) : parℓ_type S φ i ↪ ℕ
                        • Gen_α (i : Fin (k + 1)) : Generator.ProximityGenerator (BlockRelDistance.indexPowT S φ ↑i) F
                        • inst6 (i : Fin (k + 1)) : Fintype (Gen_α i).parℓ
                        • BStar (i : Fin (k + 1)) : Set (BlockRelDistance.indexPowT S φ ↑i → F) → Type → NNReal
                        • errStar (i : Fin (k + 1)) : Set (BlockRelDistance.indexPowT S φ ↑i → F) → Type → ℝ → ENNReal
                        • h (i : Fin (k + 1)) : MutualCorrAgreement.hasMutualCorrAgreement (Gen_α i) (↑(BStar i (↑(Gen_α i).C) (Gen_α i).parℓ)) (errStar i (↑(Gen_α i).C) (Gen_α i).parℓ)
                        • hcard (i : Fin (k + 1)) : Fintype.card (Gen_α i).parℓ = 2
                        • hcard' (i : Fin (k + 1)) : Fintype.card (parℓ_type S φ i) = 2

                          theorem Fold.folding_listdecoding_if_genMutualCorrAgreement {F : Type} [Field F] [Fintype F] [DecidableEq F] {ι : Type} [Fintype ι] [Pow ι ℕ] [DecidableEq ι] {S : Finset ι} {φ : ι ↪ F} [Smooth φ] {k m : ℕ} {S' : Finset (BlockRelDistance.indexPowT S φ 0)} {φ' : BlockRelDistance.indexPowT S φ 0 ↪ F} [(i : ℕ) → Fintype (BlockRelDistance.indexPowT S φ i)] [DecidableEq (BlockRelDistance.indexPowT S φ 0)] [Smooth φ'] [h : {f : BlockRelDistance.indexPowT S φ 0 → F} → BlockRelDistance.DecidableBlockDisagreement 0 k f S' φ'] [(i : ℕ) → Neg (BlockRelDistance.indexPowT S φ i)] {C : Set (BlockRelDistance.indexPowT S φ 0 → F)} (hcode : C = ↑(smoothCode φ' m)) (hLe : k ≤ m) {δ : NNReal} {params : GenMutualCorrParams S φ k} :
                          have x := GenMutualCorrParams.inst1; have x := ⋯; (do let αs ← PMF.uniformOfFintype (Fin k → F) pure (∀ (f : BlockRelDistance.indexPowT S φ 0 → F), (0 < δ ∧ δ < 1 - Finset.univ.sup fun (j : Fin (k + 1)) => GenMutualCorrParams.BStar j (↑(GenMutualCorrParams.Gen_α j).C) (GenMutualCorrParams.Gen_α j).parℓ) → have listBlock := BlockRelDistance.listBlockRelDistance 0 k f S' C hcode δ; have fold := fold_k f αs hLe; have foldSet := fold_k_set listBlock αs hLe; let kFin := ⟨k, ⋯⟩; have Cₖ := (GenMutualCorrParams.Gen_α kFin).C; have listHamming := ListDecodable.relHammingBall (↑Cₖ) fold ↑δ; foldSet ≠ listHamming)) True < ∑ i : Fin (k + 1), GenMutualCorrParams.errStar i (↑(GenMutualCorrParams.Gen_α i).C) (GenMutualCorrParams.Gen_α i).parℓ ↑δ

                          Theorem 4.20 Let C = RS[F,ι,m] be a smooth ReedSolomon code For k ≤ m and 0 ≤ i ≤ k, let Cⁱ = RS[F,ι^(2ⁱ),m-i] and let Gen(2,α) be a proxmity generator with mutual correlated agreement for C⁰,...,C^{k} with proximity bounds BStar and errStar Then for every f : ι → F and δ ∈ (0, 1 - max {i ∈ [0,k]} BStar(Cⁱ, 2)) Pr_{α ← F} [ fold_k_set(Λᵣ(0,k,f,S',C,hcode,δ),vecα) ≠ Λ(Cᵏ,fold_k(f,vecα),δ)] < ∑ i ∈ [0,k] errStar(Cⁱ,2,δ), where fold_k_set and fold_k are as defined above, vecα is generated from α as {1,α,α²,..} Λᵣ(0,k,f,S',C,hcode,δ) corresponds to the list of codewords of C δ-close to f, wrt (0,k)-wise block relative distance. Λ(Cᵏ,fold(f,vecα),δ) is the list of codewords of Cᵏ δ-close to foldf(f, vecα), wrt the relative Hamming distance Below, we use an instance of the class GenMutualCorrParams to capture the conditions of proxmity generator with mutual correlated agreement for codes C⁰,...,C^{k}.

                          (do let α ← PMF.uniformOfFintype F pure (∀ {f : BlockRelDistance.indexPowT S φ 0 → F}, 0 < δ ∧ δ < 1 - BStar C' 2 → have listBlock := BlockRelDistance.listBlockRelDistance 0 k f S_0 C hcode δ; have vec_α := fun (x : Fin 1) => α; have foldSet := fold_k_set listBlock vec_α hm; have fold := fold_k f vec_α hm; have listBlock' := BlockRelDistance.listBlockRelDistance 1 k fold S_1 C' hcode' δ; foldSet ≠ listBlock')) True < ↑(errStar C' 2 δ)

                          Lemma 4.21 Let C = RS[F,ι,m] be a smooth ReedSolomon code and k ≤ m Denote C' = RS[F,ι^2,m-1], then for every f : ι → F and δ ∈ (0, 1 - BStar(C',2)) Pr_{α ← F} [ fold_k_set(Λᵣ(0,k,f,S_0,C,δ),α) ≠ Λᵣ(1,k-1,foldf(f,α),S_1,C',δ) ] < errStar(C',2,δ) where foldf(f,α) returns a function ι^2 → F, S_0 and S_1 denote finite sets of elements of type ι and ι², and Λᵣ denotes the list of δ-close codewords wrt block relative distance. Λᵣ(0,k,f,S_0,C) denotes Λᵣ at f : ι → F for code C and Λᵣ(1,k,foldf(f,α),S_1,C') denotes Λᵣ at foldf : ι^2 → F for code C'.

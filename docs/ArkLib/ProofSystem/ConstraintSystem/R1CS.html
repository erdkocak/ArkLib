    Rank-1 Constraint System (R1CS) #

    This file defines the R1CS (Rank-1 Constraint System) relation

    • The definition is in terms of Fin vectors and matrices. In the future, we may consider more efficient representations such as Vector and Vector m (Vector n Œ±).
    • We define padding (on the right) for R1CS instances, and show that padding preserves the R1CS relation.

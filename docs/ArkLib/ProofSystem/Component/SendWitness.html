    Simple Oracle Reduction - SendWitness #

    This file contains the (oracle) reduction for the trivial one-message protocol where the prover sends the (entire) witness to the verifier. There are two variants:

    1. For oracle reduction: the witness is an indexed family of types, and sent in a single oracle message to the verifier (using the derived indexed product instance for oracle interface).

    We also define a simpler variant where one sends a single witness (converted to be indexed by Fin 1).

    1. For reduction: the witness is a type, and sent as a statement to the verifier.

    First, the reduction version (no oracle statements)

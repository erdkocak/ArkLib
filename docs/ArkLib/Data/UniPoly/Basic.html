    Computable Univariate Polynomials #

    This file contains a computable datatype for univariate polynomial, UniPoly R. This is internally represented as an array of coefficients.

    source
    @[reducible, inline, specialize #[]]
    def UniPoly (R : Type u_1) :
    Type u_1

    A type analogous to Polynomial that supports computable operations. This defined to be a wrapper around Array.

    For example the Array #[1,2,3] represents the polynomial 1 + 2x + 3x^2. Two arrays may represent the same polynomial via zero-padding, for example #[1,2,3] = #[1,2,3,0,0,0,...].

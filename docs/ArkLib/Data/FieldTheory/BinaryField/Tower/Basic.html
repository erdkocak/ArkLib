    Binary Tower Fields #

    Define the binary tower field GF(2^{2^k}) as an iterated quadratic extension of GF(2).

    Main Definitions #

    • BTField k : the binary tower field GF(2^{2^k}) as an iterated quadratic extension of GF(2), where BTField 0 = GF(2)

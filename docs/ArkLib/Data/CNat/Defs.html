    The Cayley Tower over the Natural Numbers #

    This file implements a general framework for the Cayley transformation that generalizes the AssocNat construction. The transformation takes a type T with a successor operation and produces Cayley T where addition is definitionally associative.

    By iteratively applying this transformation, we create the CNat hierarchy where higher-order operations gain definitional associativity properties.

    source
    structure Cayley (T : Type u) [HasSucc T] :
    Type u

    The Cayley T type is a structure containing an endomorphism on T along with a proof that this function commutes with succ. This is the key property that enables definitional associativity.

    • toFun : T → T

      The underlying endomorphism on T.

    • toFun_succ (t : T) : self.toFun (succ' t) = succ' (self.toFun t)

      The proof that the function commutes with succ. This is the key property that enables definitional associativity.

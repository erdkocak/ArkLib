    HasSucc Typeclass #

    This file defines the HasSucc typeclass for types that have a successor operation, along with the LawfulHasSucc class that ensures the successor operation behaves correctly with respect to addition.

    Main Definitions #

    • HasSucc T: A typeclass for types with a successor operation succ : T → T
    • LawfulHasSucc T: A typeclass ensuring that succ x = x + 1

    Implementation Notes #

    The LawfulHasSucc class requires that the type has Add and One instances, and that the successor operation is equivalent to adding one on the right.

      This class ensures that succ x = x + 1 for all x : T. It requires the type to have Add and One instances.

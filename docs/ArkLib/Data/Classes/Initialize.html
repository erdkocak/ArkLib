    class Initialize (α : Type u) (β : Type v) :
    Type (max u v)

    Type class for types that can be initialized from a value of type β.

    This abstraction allows different types to be initialized from the same input type, providing a uniform interface for construction. In the context of cryptographic sponges, this is typically used to initialize states from a seed or initialization vector.

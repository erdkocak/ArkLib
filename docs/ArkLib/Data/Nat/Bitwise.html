    def Nat.getBit (k n : ℕ) :
    ℕ

    Returns the k-th least significant bit of a natural number n as a natural number (in {0, 1}).

    We decompose each number j < 2^ℓ into its binary representation : j = Σ k ∈ Fin ℓ, jₖ * 2ᵏ

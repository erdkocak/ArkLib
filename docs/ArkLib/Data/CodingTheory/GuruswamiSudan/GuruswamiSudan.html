    structure GuruswamiSudan.Condition {F : Type} [Field F] [DecidableEq F] {n : ℕ} (k r D : ℕ) (ωs : Fin n ↪ F) (f : Fin n → F) (Q : Polynomial (Polynomial F)) :
    Prop

    Guruswami–Sudan conditions for the polynomial searched by the decoder.

    These conditions characterize the existence of a nonzero bivariate polynomial Q(X,Y) that vanishes with sufficiently high multiplicity at all interpolation points (ωs i, f i). As in the Berlekamp–Welch case, this can be shown to be equivalent to solving a system of linear equations.

    Parameters:

    • k : ℕ — Message length parameter of the code.
    • r : ℕ — Multiplicity parameter; controls how many derivatives of Q must vanish at each interpolation point.
    • D : ℕ — Degree bound for Q under the weighted degree measure.
    • ωs : Fin n ↪ F — The domain of evaluation.
    • f : Fin n → F — Received word (evaluation of the encoded polynomial, possibly corrupted).
    • Q : Polynomial (Polynomial F) — The candidate bivariate polynomial in variables X and Y.
    • Q_ne_0 : Q ≠ 0

      Q ≠ 0

    • Q_deg : Polynomial.Bivariate.weightedDegree Q 1 (k - 1) ≤ ↑D

      Degree of the polynomial.

    • Q_roots (i : Fin n) : Polynomial.eval (ωs i) (Polynomial.eval (Polynomial.C (f i)) Q) = 0

      (ωs i, f i) must be root of the polynomial Q.

    • Q_multiplicity (i : Fin n) : some r ≤ Polynomial.Bivariate.rootMultiplicity Q (ωs i) (f i)

      Multiplicity of the roots is at least r.

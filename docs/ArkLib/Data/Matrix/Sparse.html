    Sparse representation of matrices #

    source
    structure SparseMatrix (m : Type u_1) (n : Type u_2) (α : Type u_3) :
    Type (max (max u_1 u_2) u_3)

    The sparse representation of a matrix m → n → α consists of:

    • The number of non-zero entries k : ℕ
    • The row indices row : Fin k → m
    • The column indices col : Fin k → n
    • The values val : Fin k → α

    This representation is not unique. In particular, we may have duplicate (row, col) pairs, and some val may be zero.

    • numEntries : ℕ
    • row : Fin self.numEntries → m
    • col : Fin self.numEntries → n
    • val : Fin self.numEntries → α

    Conversion of Univariate polynomials to Multilinear polynomials #

    Univariate polynomials of degree < 2ᵐ can be writen as degree wise linear m-variate polynomials by ∑ aᵢ Xⁱ → ∑ aᵢ ∏ⱼ Xⱼ^(bitⱼ(i))

    source

    Fin m →₀ ℕ

    Given integers m and i this computes monomial exponents ( σ(0), ..., σ(m-1) ) = ( bit₀(i), ..., bitₘ₋₁(i) ) such that we have X_0^σ(0)⬝ ⋯ ⬝ X_(m-1)^σ(m-1). For i ≥ 2ᵐ this is the bit reprsentation of (i mod 2ᵐ)
